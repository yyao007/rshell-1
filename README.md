rshell
==========

---
Project Source can be downloaded from https://github.com/richardheng/rshell.git

How to run file
------------

```
git clone http://github.com/richardheng/rshell.git
cd rshell
git checkout hw0
make
bin/rshell
```

-----
Overview

To demostrate the basic abilities of a command shell. 

Known Bugs
